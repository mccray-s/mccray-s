## 📱 Featured Project: Unraid Deck

**Unraid Deck** is the ultimate native mobile companion for your Unraid server. It brings real-time monitoring, powerful Docker & VM management, and interactive widgets directly to your Apple devices.

**✨ Key Features:**
*   **100% Native Experience:** Built with SwiftUI for fluid animations, haptic feedback, and Dark Mode support.
*   **Privacy First Architecture:** 100% Direct Connection to your server (no middleman or third-party relay servers). Your API keys are strictly encrypted via Apple's Secure iOS Keychain.
*   **Powerful Management:** Stream Docker live logs, execute batch operations for containers and VMs, and upload custom icons.
*   **Interactive Widgets:** Monitor CPU, Memory, and Array usage directly from your iOS Home Screen, and control Docker containers without opening the app.
*   **Network Friendly:** Full support for Tailscale, Self-Signed Certificates, and custom HTTP headers for reverse proxies like Authelia.
